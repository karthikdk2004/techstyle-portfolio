# D. Karthik Reddy — Portfolio

**Live site**: [karthikreddy-techstyle-portfolio.vercel.app](https://karthikreddy-techstyle-portfolio.vercel.app/)

Personal portfolio built with Vite, React, TypeScript, and Tailwind CSS. Deployed on Vercel.

## Tech stack

- Vite
- React 18 + TypeScript
- Tailwind CSS
- shadcn/ui

## Running locally

```sh
git clone https://github.com/karthikdk2004/techstyle-portfolio.git
cd techstyle-portfolio
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080).

## Building for production

```sh
npm run build
```

Output goes to `dist/`.
